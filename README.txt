# 📚 Mini Sistema de Estoque de Livros (PHP + PostgreSQL)

Este projeto é um mini sistema web desenvolvido em PHP com PDO e PostgreSQL, que permite gerenciar um estoque de livros e suas editoras. Inclui dois CRUDs completos e uma interface responsiva em HTML e CSS puro.

## 🚀 Funcionalidades

- Cadastro, edição e exclusão de livros 📘
- Cadastro, edição e exclusão de editoras 🏢
- Relacionamento entre livros e editoras
- Interface moderna e responsiva
- Banco de dados PostgreSQL

## 🛠️ Tecnologias

- PHP 8+
- PostgreSQL
- HTML5, CSS3
- PDO (PHP Data Objects)
